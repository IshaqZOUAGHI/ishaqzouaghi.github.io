Vous trouverez ici une sélection de liens pratiques autour des technologies Java qui m'ont particulièrement intéressées cette semaine 02 (janvier 2011).

Eclipse  

* Un [billet](http://eclipsedriven.blogspot.com/2010/12/types-of-applications-you-can-create.html) qui présente les types d'application qui peuvent être développés avec la plateforme Eclipse.
* Un [article](http://blog.xebia.fr/2010/12/29/utiliser-guice-et-peaberry-pour-developper-un-plugin-eclipse/) intéressant qui présente comment utiliser l'injection de dépendance avec [GUICE](http://code.google.com/p/google-guice/) et [Peaberry](http://code.google.com/p/peaberry/), sa version pour OSGi, pour le développement de plugin Eclipse.  
* Orion un [nouveau projet](http://dev.eclipse.org/blogs/mike/2011/01/11/introducing-orion/) de la fondation Eclipse a pour objectif de fournir un environnement de développement dans le navigateur ([wiki](http://wiki.eclipse.org/Orion) du projet).

Java  

* HUDSON, l'outil d'intégration continue change de nom pour Jenkins, les raisons sont évoquées dans ce [billet](http://hudson-labs.org/content/hudsons-future).  

Divers  

* Un [billet](http://blog.loof.fr/2011/01/rpcencoded-avec-jax-ws.html) qui décrit comment manipuler des services web étendus (WSDL) en mode RPC/Encoded avec JAX-WS. Pour rappel, ce mode n'est plus supporté par la version actuelle.
* La plateforme de Forge Redmine 1.1 est [disponible](http://www.redmine.org/projects/redmine/wiki/Download).
